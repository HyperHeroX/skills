# QA Engineer Technical Capabilities Reference

> Source: Adapted from `sample/unit-testing/agents/test-automator.md`

## Test-Driven Development (TDD) Excellence

- **Test-first development**: Red-green-refactor cycle automation
- **Failing test generation**: Verification for proper TDD flow
- **Minimal implementation guidance**: Passing tests efficiently
- **Refactoring test support**: Regression safety validation
- **TDD cycle metrics**: Cycle time and test growth tracking
- **Chicago School (state-based)** and **London School (interaction-based)** TDD approaches
- **Property-based TDD**: Automated property discovery and validation

## AI-Powered Testing Frameworks

- **Self-healing test automation**: Testsigma, Testim, Applitools
- **AI-driven test case generation**: Natural language processing
- **Machine learning**: Test optimization and failure prediction
- **Visual AI testing**: UI validation and regression detection
- **Predictive analytics**: Test execution optimization
- **Intelligent test data generation**: Automated management

## Modern Test Automation Frameworks

- **Cross-browser automation**: Playwright and Selenium WebDriver
- **Mobile test automation**: Appium, XCUITest, Espresso
- **API testing**: Postman, Newman, REST Assured, Karate
- **Performance testing**: K6, JMeter, Gatling
- **Contract testing**: Pact and Spring Cloud Contract
- **Accessibility testing**: axe-core and Lighthouse
- **Database testing**: Validation frameworks

## CI/CD Testing Integration

- **Pipeline integration**: Jenkins, GitLab CI, GitHub Actions
- **Parallel test execution**: Test suite optimization
- **Dynamic test selection**: Based on code changes
- **Containerized testing**: Docker and Kubernetes environments
- **Test result aggregation**: Reporting across platforms
- **Progressive testing**: Canary deployments

## Performance and Load Testing

- **Scalable load testing**: Cloud-based execution architectures
- **Performance monitoring**: APM integration during testing
- **Stress testing**: Capacity planning validation
- **API performance testing**: SLA validation
- **Mobile app performance**: Cross-device testing
- **Real user monitoring (RUM)**: Synthetic testing

- **Database state management**: Cleanup automation
- **Environment-specific provisioning**: Test data management

## Quality Engineering Strategy

- **Risk-based testing**: Coverage analysis

- **Exploratory testing integration**: Automation complement
- **Quality metrics**: KPI tracking systems
