# 職位名稱：資深測試工程師（Senior QA Engineer / Test Engineer）

## 📌 角色定位

資深測試工程師負責建立完整的測試策略與流程，涵蓋功能測試、自動化測試與性能測試等領域。此角色在產品品質保證（QA）中扮演關鍵角色，確保交付系統具備高穩定性、可用性與可維護性，並協助團隊導入測試自動化與品質文化。

---

## 🛠️ 核心技能要求

### 1. 測試策略與規劃能力
- 能撰寫測試計畫書、測試案例（Test Cases）、測試報告（Test Report）
- 熟悉測試類型：單元測試、整合測試、回歸測試、系統測試、UAT

### 2. 測試方法與流程
- 熟悉 STLC（Software Testing Life Cycle）與缺陷管理流程
- 使用如 Jira、TestRail、Xray 進行測試規劃與追蹤

### 3. 缺陷管理與分析
- 能準確回報與分類缺陷，並撰寫重現步驟與建議
- 熟悉 Bug Lifecycle 與測試結果追蹤

### 4. 自動化測試能力
- 熟悉 Selenium、Playwright、Cypress、Appium 等測試框架
- 能設計維護可擴充的自動化測試腳本（UI / API / E2E）

### 5. API 與資料驗證
- 熟悉使用 Postman / Swagger 進行 API 測試與驗證
- 能操作 SQL 查詢驗證資料準確性與整合性

---

## 🧩 在專案中的工作描述

- 依據產品需求撰寫測試計畫、案例與執行手冊
- 執行功能測試、自動化測試與回歸測試，確保品質一致
- 使用 Bug Tracking 工具提交與追蹤缺陷，與開發團隊協作解決
- 協助進行 API 測試、資料驗證與多端裝置相容性測試
- 定期產出測試報告與風險分析，提供開發/產品團隊參考
- 參與 Sprint/Release Meeting，提供測試回饋與品質建議
- 主導測試自動化策略與測試環境建置

---

## ⭐ 加強技能（進階加分條件）

### 測試自動化進階實作
- 熟悉 CI/CD 測試整合流程（如 GitHub Actions、GitLab CI、Azure DevOps）
- 熟悉 BDD 測試方式與框架（如 Cucumber、SpecFlow）

### 效能與壓力測試
- 熟悉 JMeter、k6、Locust 等性能測試工具
- 能分析 TPS、RT、Memory、CPU 使用率與效能瓶頸

### 安全性與合規測試
- 熟悉 OWASP Top 10 與基本滲透測試技巧
- 具備 ISO 27001、GDPR、PCI-DSS 測試經驗者加分

### 多端測試經驗
- 熟悉 Web、Mobile App（Android/iOS）、桌面應用程式等跨平台測試策略
- 能使用模擬器與真機驗證不同裝置的操作一致性

### 測試流程優化與品質文化推動
- 曾主導導入測試流程、質量門檻與品質指標（如 Defect Leakage Rate）
- 具備跨團隊溝通能力，能推動測試左移（Shift-Left Testing）
