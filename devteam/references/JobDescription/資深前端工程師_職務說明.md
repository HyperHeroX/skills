# 職位名稱：資深前端開發工程師（Senior Frontend Developer）

## 📌 角色定位

資深前端開發工程師負責規劃與實作前端架構、開發模組化元件、提升使用者體驗（UX）與效能表現（Performance）。此角色需具備獨立設計與開發中大型前端應用的能力，並能協助團隊制定技術標準與最佳實務。

---

## 🛠️ 核心技能要求

### 1. 前端框架熟練度
- 精通 Vue.js、React 或 Angular 等現代前端框架
- 熟悉組件化開發、狀態管理（如 Vuex、Pinia、Redux）

### 2. HTML / CSS / JavaScript 能力
- 精通語意化 HTML5、CSS3（含 SCSS/SASS、Tailwind）
- 熟悉 JavaScript ES6+ 與 TypeScript 開發語法與模組化

### 3. API 串接與資料處理
- 熟悉 RESTful API、Axios、Fetch 串接與錯誤處理
- 具備基本資料結構與非同步處理觀念（Promise, Async/Await）

### 4. 前端建構與工具鏈
- 熟悉 Vite、Webpack、ESBuild、Babel 等建構工具
- 熟悉 NPM、Yarn、PNPM 等套件管理工具

### 5. 測試與除錯能力
- 能撰寫單元測試與端對端測試（如 Vitest、Jest、Cypress）
- 熟悉前端除錯技巧與瀏覽器 DevTools 使用

---

## 🧩 在專案中的工作描述

- 撰寫並維護核心模組與元件，實作畫面互動與資料綁定
- 與後端協作整合 API，負責資料流程與畫面狀態管理
- 協助 UI/UX 設計師轉換設計稿為互動網頁介面（如 Figma to Code）
- 建立與維護前端專案架構與模組規範
- 優化網站效能（LCP/FID/CLS）、提升 SEO 與 Lighthouse 分數
- 協助團隊進行 Code Review、知識傳承與技術導入建議
- 撰寫技術文件與使用者操作說明

---

## ⭐ 加強技能（進階加分條件）

### 架構與工程實務
- 熟悉 SPA / MPA / PWA 架構與技術選型
- 熟悉 SSR（如 Nuxt、Next.js）或 SSG 靜態網站生成

### DevOps 與版本控制
- 熟悉 Git Flow、PR 流程與 CI/CD 自動化部署（如 GitHub Actions、Vercel、Netlify）
- 能設計部署流程並處理版本切換、分支策略

### 測試與自動化
- 熟悉自動化測試框架與整合測試流程（CI test matrix）
- 能撰寫 UI snapshot、Mock API 測試與 e2e 測試流程

### 效能與可用性最佳化
- 熟悉 Core Web Vitals、Lazy Loading、Tree Shaking、Code Splitting
- 具備 Web Accessibility（a11y）與 i18n 多語系開發經驗

### 團隊技術領導力
- 能主導元件規範制定、前端架構選型與 refactor 計畫
- 具備技術講解與內部訓練能力，提升團隊開發品質
