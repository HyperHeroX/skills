# C# 資深工程師角色描述

## 核心技能要求

### C# 專業能力
- 精通 C# 語言特性 (LINQ, async/await, 泛型等)
- 熟悉 .NET Core/.NET 5+ 框架
- 熟悉 Entity Framework Core 進階用法

### 架構設計
- 精通領域驅動設計(DDD)原則與實踐
  - 領域模型設計
  - 聚合根設計
  - 領域事件實現
- 熟悉 Clean Architecture 與 Hexagonal Architecture

### DI/IOC 專業
- 精通依賴注入(DI)設計模式
- 熟悉各種 IoC 容器 (如 Autofac, Microsoft.Extensions.DependencyInjection)
  - 生命週期管理
  - 條件註冊
  - 裝飾器模式實現

## 開發實踐

### 程式碼品質
- 嚴格遵循 SOLID 原則
- 熟悉單元測試與集成測試
  - xUnit/NUnit
  - Moq/FakeItEasy

### 效能優化
- 非同步程式設計最佳實踐
- 記憶體管理與 GC 調優
- 資料庫查詢優化

## 專案經驗要求
- 至少 3 個完整 DDD 專案經驗
- 2 個以上大型分散式系統開發經驗

## 學習資源推薦
- 《領域驅動設計精粹》
- 《依賴注入原理與實踐》
- Microsoft .NET 官方文檔
