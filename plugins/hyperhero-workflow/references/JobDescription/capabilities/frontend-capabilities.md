## Core React/Framework Expertise

- **React 19+ features**: Server Components, Actions, async transitions
- **Concurrent rendering**: Suspense patterns for optimal UX
- **Advanced hooks**: useActionState, useOptimistic, useTransition, useDeferredValue
- **Component architecture**: Performance optimization (React.memo, useMemo, useCallback)
- **Custom hooks**: Hook composition patterns
- **Error boundaries**: Error handling strategies

## Next.js & Full-Stack Integration

- **App Router**: Server Components and Client Components
- **React Server Components (RSC)**: Streaming patterns
- **Server Actions**: Seamless client-server data mutations
- **Advanced routing**: Parallel routes, intercepting routes, route handlers
- **ISR**: Incremental Static Regeneration and dynamic rendering
- **Image optimization**: Core Web Vitals optimization

## Modern Frontend Architecture

- **Component-driven development**: Atomic design principles
- **Micro-frontends**: Module federation architecture
- **Design system integration**: Component libraries
- **Build optimization**: Webpack 5, Turbopack, Vite
- **Bundle analysis**: Code splitting strategies
- **PWA implementation**: Service workers and offline-first patterns

## State Management & Data Fetching

- **Modern state management**: Zustand, Jotai, Valtio, Pinia
- **Server state**: TanStack Query, SWR for data fetching and caching
- **Context API optimization**: Provider patterns
- **Real-time data**: WebSockets and Server-Sent Events
- **Optimistic updates**: Conflict resolution

## Styling & Design Systems

- **Tailwind CSS**: Advanced configuration and plugins
- **CSS-in-JS**: emotion, styled-components, vanilla-extract
- **Design tokens**: Theming systems
- **Responsive design**: Container queries, CSS Grid, Flexbox mastery
- **Animation libraries**: Framer Motion, React Spring
- **Dark mode**: Theme switching patterns

## Performance & Optimization

- **Core Web Vitals**: LCP, FID, CLS optimization
- **Code splitting**: Dynamic imports
- **Image optimization**: Lazy loading, WebP, srcset
- **Font optimization**: Variable fonts
- **Memory leak prevention**: Performance monitoring
- **Bundle analysis**: Tree shaking, critical resource prioritization

## Testing & Quality Assurance

- **React Testing Library**: Component testing
- **Jest configuration**: Advanced testing patterns
- **E2E testing**: Playwright and Cypress
- **Visual regression testing**: Storybook, Chromatic
- **Performance testing**: Lighthouse CI
- **Accessibility testing**: axe-core
- **Type safety**: TypeScript 5.x features

## Accessibility & Inclusive Design

- **WCAG 2.1/2.2 AA compliance**: Implementation patterns, Section 508, EN 301 549
- **ARIA patterns**: Roles, states, properties for custom components, live regions
- **Keyboard navigation**: Tab order, focus trapping, skip links, roving tabindex
- **Screen reader optimization**: NVDA, JAWS, VoiceOver, TalkBack testing
- **Color contrast**: WCAG AA (4.5:1), AAA (7:1), color blindness considerations
- **Inclusive design principles**: Universal usability, cognitive accessibility
- **Assistive technology**: Voice control, switch access, screen magnification support
- **Automated testing**: axe-core, WAVE, Lighthouse, jest-axe integration

## UI Design & Visual Systems

- **Atomic design methodology**: Atoms, molecules, organisms, templates, pages
- **Component states**: Default, hover, active, focus, disabled, error, loading
- **Layout systems**: CSS Grid, Flexbox, responsive grids, container queries
- **Visual fundamentals**: Color theory, typography systems, iconography, shadows
- **Design tokens**: CSS custom properties, theming systems, dark mode
- **Design-to-code**: Tailwind CSS, CSS-in-JS, CSS Modules, SVG optimization
- **Prototyping**: Wireframing, high-fidelity prototyping, interaction design
- **Animation principles**: Micro-interactions, transitions, skeleton loaders
