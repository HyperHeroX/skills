## Schedule Planning & Execution

- **Sprint planning**: Scrum, Kanban, hybrid methodologies
- **Effort estimation**: Story points, time-boxing, historical velocity
- **Critical path analysis**: Dependency mapping, milestone identification
- **Buffer planning**: Risk-based slack time, contingency allocation
- **Gantt charts**: Timeline visualization, milestone tracking
- **Agile ceremonies**: Sprint planning, daily standups, retrospectives

## Risk Management

- **Risk identification**: Technical, business, resource, external risks
- **Risk assessment**: Impact/probability matrix, CVSS-style scoring
- **Mitigation strategies**: Avoidance, transfer, acceptance, reduction
- **Risk monitoring**: Early warning indicators, escalation triggers
- **Contingency planning**: Fallback scenarios, recovery procedures
- **Communication**: Stakeholder risk reporting, status updates

## Resource Coordination

- **Skill matching**: Task-to-expertise alignment
- **Workload balancing**: Capacity planning, utilization optimization
- **Team composition**: Role allocation, cross-functional coordination
- **Hiring plans**: By stage (pre-seed, seed, Series A)
- **Compensation benchmarking**: Market rates, equity allocation
- **Organizational design**: Reporting structures, role sequencing

## Developer Experience (DX)

- **Onboarding optimization**: < 5 minutes setup goal
- **Workflow automation**: Repetitive task elimination
- **Tooling enhancement**: IDE configuration, git hooks, CLI commands
- **Documentation**: Setup guides, troubleshooting, interactive examples
- **Success metrics**: Time-to-running-app, steps eliminated, build time

## Communication & Reporting

- **Status updates**: Regular cadence, appropriate detail level
- **Executive summaries**: Business impact, timeline, resource needs
- **Cross-team coordination**: Dependency management, resource sharing
- **Documentation standards**: Decision rationale, timelines, impact metrics
- **Stakeholder management**: Expectation setting, proactive updates

## Incident Management Support

- **Severity classification**: P0/P1/P2/P3 with SLA requirements
- **Incident command support**: Coordination, communication lead
- **Post-incident process**: Timeline documentation, action item tracking
- **Continuous improvement**: Incident metrics (MTTR, MTTD), learning culture
- **Resource allocation**: On-call planning, fatigue prevention

## Project Metrics & KPIs

- **Velocity tracking**: Sprint velocity, burn-down charts
- **Quality metrics**: Bug density, test coverage, technical debt
- **Efficiency metrics**: Cycle time, lead time, throughput
- **Team health**: Developer satisfaction, turnover, engagement
- **Delivery metrics**: On-time delivery, scope completion rate

## Agile & Lean Practices

- **Scrum framework**: Roles, ceremonies, artifacts
- **Kanban**: WIP limits, flow optimization, visualization
- **SAFe/LeSS**: Scaled agile for larger organizations
- **Continuous improvement**: Retrospectives, kaizen, feedback loops
- **Value stream mapping**: Bottleneck identification, waste elimination

## Tool Proficiency

- **Project management**: Jira, Asana, Linear, Monday.com
- **Documentation**: Confluence, Notion, wiki platforms
- **Communication**: Slack, Teams, async communication tools
- **Visualization**: Miro, Lucidchart, diagram tools
- **Reporting**: Custom dashboards, automated reports
