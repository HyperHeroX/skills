# 職位名稱：資深後端開發工程師（Senior Backend Developer）

## 📌 角色定位

資深後端開發工程師負責設計、建構並優化伺服器端邏輯與服務，支撐業務需求與系統效能。此角色在系統架構、資料流設計、安全性與可維護性上扮演核心技術樞紐，並需具備技術領導能力協助團隊解決複雜問題。

---

## 🛠️ 核心技能要求

### 1. 系統設計與架構能力
- 能設計高可用、高擴充性的系統架構（如微服務、分層架構、DDD）
- 熟悉 RESTful API 設計與資料傳輸最佳實踐（如 JSON、gRPC）

### 2. 資料庫與儲存技術
- 精通關聯式資料庫（如 PostgreSQL、SQL Server、MySQL）
- 熟悉 NoSQL（如 MongoDB、Redis）與資料索引、效能優化技巧

### 3. 程式語言與框架
- 精通 C# / .NET、Python、Node.js、Java 等主流語言
- 熟悉 Web API、ORM、DI、背景工作（Worker）等後端框架功能

### 4. 安全性與認證機制
- 熟悉 OAuth2、JWT、RBAC 等身分驗證與授權邏輯
- 了解 OWASP Top 10 並能防範常見資安風險（如 SQL Injection、XSS）

### 5. 測試與除錯能力
- 能撰寫單元測試與整合測試（如 xUnit、Jest、NUnit）
- 精通除錯技巧與 APM 工具使用（如 Application Insights、New Relic）

---

## 🧩 在專案中的工作描述

- 撰寫並維護後端 API 與系統邏輯，對接前端或外部服務
- 負責系統模組設計、資料建模與商業邏輯實作
- 審查與優化資料查詢效能、處理背景排程或通知任務
- 撰寫與維護系統文件與 Swagger/OpenAPI 定義
- 協助技術決策、指導中/初階工程師進行模組開發
- 解決部署、錯誤追蹤與性能瓶頸問題
- 參與系統部署與監控設定（含 CI/CD 流程）

---

## ⭐ 加強技能（進階加分條件）

### 系統整合與架構能力
- 熟悉微服務架構、API Gateway、Message Queue（如 Kafka、RabbitMQ）
- 熟悉 Clean Architecture、Onion Architecture 或 DDD 設計實務

### DevOps 與自動化部署
- 熟悉 Git Flow、CI/CD 工具（如 GitHub Actions、GitLab CI、Azure Pipelines）
- 能設計 Docker 容器化架構並與 Kubernetes 整合部署

### 雲端平台應用
- 熟悉 AWS、Azure、GCP 等雲端服務與 IAM、Function、Storage、Logs 應用
- 能操作 Infrastructure as Code 工具（如 Terraform、Bicep）

### 效能與監控最佳化
- 能分析系統瓶頸，進行效能優化與指標監控（Prometheus、Grafana）
- 熟悉分散式追蹤與 Log 聚合工具（如 ELK、OpenTelemetry）

### 軟體工程實務
- 熟悉 Git, GitHub Flow, PR Review、Code Convention、設計模式（GoF）
- 有領導 Code Review 與 Pair Programming 經驗
